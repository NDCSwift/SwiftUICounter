# 🔢 SwiftUI Counter

A minimal SwiftUI counter app — the classic first project for learning `@State`, `Button`, and reactive UI updates in SwiftUI.

---

## 🤔 What this is

This project is a simple tap counter that demonstrates the core SwiftUI reactivity model: how `@State` drives view re-renders when a value changes. It's intentionally minimal — one view, one state variable, increment and decrement buttons.

## ✅ Why you'd use it

- **@State** — the simplest possible example of state-driven UI
- **Button + action** — connecting user taps to state mutations
- **Automatic re-render** — shows how SwiftUI updates the view when state changes
- **Starting point** — great template to fork and build on for any counter or tally app

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/SwiftUICounter.git
cd SwiftUICounter
```

### 2. Open in Xcode
- Double-click `SwiftUICounter.xcodeproj`

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team**

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique identifier (e.g., `com.yourname.SwiftUICounter`).

---

## 🛠️ Notes

- If you see a code signing error, check that Team and Bundle ID are set.

## 📦 Requirements

- iOS 16+
- Xcode 15+
- Swift 5.9+
